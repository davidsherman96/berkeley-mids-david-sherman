# W200: Introduction to Data Science

## Course Overview
Provides students with a strong technical foundation for the rest of the curriculum by transitioning students into thinking and working like professional data scientists and software developers

## Learning Objectives
- Build core Python fluency by learning syntax, control flow, functions and modular programming, data structures, iteration and comprehensions, and error handling 
- Develop computational thinking by decomposing problems, structuring logic, and transforming ambiguous reguirements into code
- Introduce software engineering practices, such as Git, object-oriented programming, command line usage, and modular design
- Learn data manipulation and analysis skills through libraries such as NumPy and pandas

## Projects & Assignments
The final project required students to craft an exploratory data analysis (EDA) focused around a research question. Our group investigates UFO sightings data from the National UFO Reporting Center (NUFORC) to determine if UFO phenomena can be explained by proximity to identifiable sources such as:
- Airports (civilian air traffic)
- Military bases (military aircraft/operations)
- Space launch facilities (rocket launches and space debris)

The core research question is whether reported UFO sightings cluster geographically and temporally near these explainable phenomena, or if they represent genuinely unexplained events
