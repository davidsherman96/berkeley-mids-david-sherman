# W209: Data Visualization

## Course Overview
Focuses on the principles, techniques, and tools used to communicate data effectively through visual storytelling. Teaches students how to transform complex datasets into clear, compelling, and actionable narratives for technical and non-technical stakeholders.

## Learning Objectives
- Learn the fundamentals of common data visualization tools like D3, JavaScript, HTML/CSS, and Tableau
- Determine the best use cases for various visuals based on analytical goals, audience needs, and data structure
- Ensuring visuals are interactive through tooltips, filtering, user-driven exploration features

## Final Project
Access a public dataset and create an interactive dashboard that addresses a business problem.

Our team developed a comprehensive Tableau dashboard featuring a host of NBA metrics and visualizations. The goal was to provide insights into players' shooting efficiencies across the court and broad shooting trends over time. Data was obtained from the NBA API, particularly players and game log libraries. By player, you could see his total makes, attempts, field goal percent and how those compare to league average. Each player also had a heat map included to indicate their hot/cold zones on the court. The dashboard was deployed via a Flask app.

Please note: due to restrictions, our Tableau dashboard site was taken down upon graduation. You're welcome to view the PPT to see visuals or download the .twbx file to play around with the data.
